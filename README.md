# 📊 Python for Data Processing | freeCodeCamp

Este repositório documenta minha evolução na manipulação e estruturação de dados utilizando Python. Os scripts aqui presentes fazem parte da minha trilha de aprendizado prático no **freeCodeCamp**.

## 🎯 Foco do Projeto
O objetivo é aplicar lógica de programação para transformar dados brutos em informações estruturadas, focando em:
- **Data Cleaning:** Manipulação de strings e fatiamento (slicing).
- **Formatting:** Organização de logs e perfis de usuários através de f-strings.
- **Backend Logic:** Construção de scripts reutilizáveis e eficientes.

## 🛠️ Stack Tecnológica
- **Linguagem:** Python 3.x.
- **Ambiente:** Linux (Kali Linux/VirtualBox) para execução de scripts de sistemas.
- **Versionamento:** Git & GitHub.

## 📈 Projetos em Destaque
- `processamento_dados.py`: Script que automatiza a formatação de fichas de funcionários e gera códigos de identificação via slicing.
